
## 介绍

时间的枪是一款时间管理工具，它的特点：

1. 记录每天在每件事情上花费的时间
2. 按照周/月/年来统计所花费的时间（ 可以根据日期或者类别统计查看）
3. 通过时间轴快速查看每天花费的时间
4. 同步到服务器，不再怕数据丢失
5. 方便的操作，简洁大方的界面（Material Design）
